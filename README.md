# Belvo API Documentation

---

## 🇬🇧 English

### Overview

This repository contains automatically-generated CSV and Excel representations of Belvo's API objects. The data dictionaries are maintained in three languages (English, Spanish, and Portuguese) and are automatically updated whenever changes are made to the Belvo API.

### What's Inside

📁 **Data Dictionaries** - CSV representations of all Belvo API objects:
- Browse the [English data dictionaries](data_dictionaries/en/csv/)

Each data dictionary provides field-level documentation for API endpoints including accounts, transactions, tax returns, employment data, and more.

### Automation

This repository is automatically maintained. Content is generated and updated based on changes to the Belvo API, ensuring the documentation stays in sync with the latest API specifications.

### Contributing

Contributions to this repository are rare, as the content is automatically generated. However, if you identify an issue or would like to suggest an improvement, please note the following requirements:

**All pull requests must:**
- Be clearly documented with a detailed description of the changes and rationale
- Include properly labeled commit messages
- Be reviewed and approved by members of the Belvo team

Before submitting a PR, please ensure:
- [ ] The pull request description clearly explains what changes are being made and why
- [ ] Commit messages follow proper labeling conventions
- [ ] You understand that all contributions will be reviewed by the Belvo team
